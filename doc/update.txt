Steps required to update different versions:

1.0 -> 1.1 (using Rails 2.3.11)
==========
sudo gem install rails --no-rdoc --no-ri --version 2.3.11 --verbose

# install bundler
# requires rubygems >= 1.3.6
sudo gem update --system -V
sudo gem install bundler --no-ri --no-rdoc --verbose

cp local_settings.yml -> config

# deploy using cap

# after deploy, restart nginx to avoid
# undefined method `load_yaml' for Gem:Module
# see http://destiney.com/blog/undefined-method-load-yaml-for-gem-module

# move the log files to the shared log directory
sudo vim /etc/kannel/kannel.conf

log-file = "/var/www/ocp/shared/log/kannel.log"

group = smsc
...
log-file = "/var/www/ocp/shared/log/modem.log"

group = smsbox
...
log-file = "/var/www/ocp/shared/log/smsbox.log"

In order to use this permission needs to be created:

# kannel needs to be in opencellpager group to be able to write to the files there
sudo usermod -a -G opencellpager kannel

# opencellpager needs to be in the kannel group in order to be able to read the log files (using tail)
sudo usermod -a -G kannel opencellpager

# you can check the groups a user is in by doing the following:
id kannel
id opencellpager
