Scope of this project

The Project is a pilot project designed to determine if a cell-phone paging system will be useful and usable at your hospital. This system is designed to allow health care workers (including doctors, nurses, administrative and laboratory staff) to communicate more effectively by being able to send pages to each other using a computer with network access and cell phones.

Limitation of the Project

The system may be used to communicate messages. However, there can be no guarantee that messages will be delivered or received. The system cannot be relied on as the only means to communicate critical information regarding patient care. The sender CANNOT rely on this as the only means to communicate. Existing means already exist within the hospital, and should be used in conjunction with this system.

By using this system, you acknowledge your agreement of these limitations.
